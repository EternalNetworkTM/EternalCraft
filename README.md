# EternalCraft

Website for EternalCraft SMP, currently work-in-progress.


# Building

```sh
$ npm install
$ npm install --global gulp-cli
$ gulp serve # or 'gulp build' to build without serving
```

Local server will be started on `http://localhost:8000`.
